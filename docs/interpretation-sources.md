# 解读体系与参考书目

本站的牌义解读引擎（`js/app.js` 解读文案生成 + `js/lore.js` 牌义知识库）
综合了塔罗领域四部公认的权威著作的方法论，全部文字为中文原创撰写：

| 书目 | 作者 | 贡献的层次 |
|---|---|---|
| 《The Pictorial Key to the Tarot》(1911，公有领域) | A. E. Waite | **牌面意象层**：`lore.js` 中每张牌的 `img` 字段，忠实描述韦特-史密斯牌的经典画面符号 |
| 《Seventy-Eight Degrees of Wisdom》（塔罗圣经《78度的智慧》） | Rachel Pollack | **心理象征层**：大阿卡纳「愚者之旅」的叙事弧线与逐张深挖（`MAJOR_TALK` 手写库的底层框架） |
| 《Learning the Tarot》 | Joan Bunning | **实用语境层**：按提问领域（感情/事业）给出可对照的具体情境，`lore.js` 的 `loveUp` / `loveRv` 字段方法论来源 |
| 《The Complete Book of Tarot Reversals》 | Mary K. Greer | **逆位读法**：逆位不是简单的「反义」，而是能量的受阻 / 内转 / 过度 / 松绑——所有逆位文案按此框架撰写 |

## 引擎的四层输出结构

每张牌的解读 = ①位置叙事引子（牌阵位置视角）→ ②牌面意象（Waite）
→ ③牌义正文（Pollack 式深挖，数字×花色织体）→ ④行动低语（Bunning 式可执行建议）。
关系牌阵额外织入感情语境层（`loveUp` / `loveRv`）。
全部翻开后输出「整体启示」：大牌占比、花色偏向、顺逆基调与牌阵故事线。
